# Bike Sales & Retail Analytics Repository

Welcome to the Bike Sales & Retail Analytics repository.

This repository contains multiple Excel-based analysis projects, pivot table labs, visualisation exercises, and supporting datasets.
It is organised into clearly separated folders to help you navigate through the different projects and resources.

---

## Repository Structure

### 1. Bike_Sales_Pivot_Table_and_Visualisations_Lab

Contains all resources related to the Bike Sales Pivot Table and Data Visualisation lab.

Includes:

- Screenshots (P1.jpg – P8.jpg, V1.jpg, V2.jpg, V3.jpg, etc.)
  
Illustrations of pivot table creation, formatting steps, and visualisation outputs.

Datasets:

- Bike_Sales_Pivot_Dataset.xlsx
- Bike_Sales_Pivot_Lab.xlsx
- Bike_Sales_Visualisations_Dataset.xlsx
- Bike_Sales_Visualisations_Lab.xlsx

README.md explaining the tasks and workflow.


### 2. Sales_Performance_Analysis

Contains files related to sales performance analysis and dynamic formulas.

Includes:

- Sales_Category_by_Sales_Volume.jpg
- Sales_Performance_Pivot_Table.jpg
- Sales_Performance_Switch_Function.xlsx

README.md explaining analysis steps and KPIs.


### 3. retail_sales_dataset_project

A comprehensive project covering sales analysis using Excel functions, pivot tables, and lookups.

Includes:

- Screenshots of key tasks (T1.1.jpg, T1.2.jpg, … VLOOKUP.jpg, XLOOKUP.jpg)
  
Used to demonstrate formulas and analysis methods.

Datasets:

- retail_sales_dataset.xlsx
- retail_sales_dataset_Project.xlsx

README.md describing the tasks and objectives.


### 4. Root Files

LICENSE – Usage rights for the repository

Root README.md – Overview of entire repo (this document)

## What This Repository Covers

This repository demonstrates practical Excel skills used in business analytics, including:

- Pivot table creation & formatting
- Data cleaning and transformation
- Sales performance analysis
- Lookup functions (VLOOKUP, XLOOKUP, INDEX/MATCH)
- Conditional formulas (SUMIFS, COUNTIF, AVERAGEIFS)
- Dashboard and chart creation
- Scenario analysis & KPIs

### How to Use This Repository

- Navigate to each project folder for datasets and screenshots.
- Follow the README inside each folder for instructions on the workflow.
- Use the datasets to practice pivot tables, formulas, and visualisation.
- Compare your results with the provided screenshots.
